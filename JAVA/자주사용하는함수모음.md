## == vs equals()
==은 주고 값을 비교하지만 equals()는 data값을 비교한다.
String은 equals()를 사용해야한다.


## charAt()

``` java
Stirng str = "String";
char s = str.charAt(0);
System.out.println(s);
// output => S;
```

String 으로 저장된 문자열 중에서 한 글자만 골라 char로 리턴해준다.

chartAt()안에 들어갈 숫자는 문자열에서 해당 글자의 인덱스 값이다.
